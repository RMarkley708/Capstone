sticky & responsive navbar

The navigation bar is sticky, responsive, and collapses into a hamburger menu when the screen size gets smaller. This is done by using CSS and JavaScript. The navbar is the first thing in my life I coded using JavaScript. If you are gentle, you might call the resulting code hacky. Maybe clumbsy is the better word to describe it.

marquee - powered by css

This marquee is animated by a few lines of CSS. The only limitation: The maximum number of charaters is determined by the container's width. Remember the internet at the dawn of the millennium?

code (link to codePen) & demo in one

While marquee elements are not commonly used in modern web design due to better alternatives, they hold a special place in web history and can evoke a sense of nostalgia for those who remember the early days of the internet.

flipping boxes

Boxes are a great way to structure and highlight content. I use boxes as the central design element of this web page. Most of my boxes are interactive flipping boxes They have a front and back side. Click & choose which side you want to see. The flaw of the flipping boxes: Making them responsive lets them be a bit jerky.
