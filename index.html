web design capstone project

Welcome to my capstone project of Michigan Univerity's Coursera specialization Web Design for Everybody: Basics of Web Development & coding. The project's objective: Design and code a responsive and interactive static web page from scratch.

capstone specs

×

design & code your site from scratch

  • create your web page from scratch - without the help of blueprints or templates
  • each page must validate as validator.w3.org
    - use proper semantic HTML tags

make your site responsive

  • create as least two viewports - desktop & mobile
  • extra: create an additional viewport for tablets

make your site interactive

  • add JavaScript to add interactivity
  • use CSS and JavaScript to animate elements
responsive interactive coded from scratch

the five courses

The five courses of the specialization offer a first overview of frontend development. They provide basic knowledge of HTML5, CSS3, and JavaScript. Nevertheless, the bulk of this web page is based on various textbooks, dissecting & breaking other people's code, and especially writing and rewriting a lot of code - learn on the fly.

web design for everybody courses

Web Design for Everybody on coursera.org

Introduction to HTML5

There is a strong emphasis on adhering to syntactic standards for validation and semantic standards to promote wide accessibility.

Introduction to CSS3

The emphasis is on learning how to write CSS rule, how to test code & how to establish good programming habits.

Interactivity with JavaScript

This is about the basics of JavaScript: React to DOM Events and dynamically alter the contents and style of a page.

Advanced Styling with Responsive Design

This course expands upon the basic knowledge of CSS3 to include topics such as wireframes, fluid design & media queries.

Web Design for Everybody Capstone

Demonstrate the ability to design and implement a responsive site for a minimum of three platforms. Adherence to validation standards is required.
